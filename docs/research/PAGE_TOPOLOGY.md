# Page topology

1. Skip link: keyboard-only until focused.
2. Support bar: teal-to-purple gradient with phone and online-support action.
3. Header: logo plus desktop navigation; collapsible navigation below 768px.
4. Hero: fixed-height photograph, centred H1, and overlapping action links.
5. Information: two equal columns on desktop, one column on mobile.
6. Footer: search, policy links, and B.C. mark; stacks on mobile.

The original header is fixed. The content reserves its height, and the mobile breakpoint is 768px.
